## What are Web APIs?

- Provided by the **runtime environment** (e.g., browser or Node.js)
- **Not part of JavaScript itself**
- Provide additional functionality that JavaScript can use

### Common features:
- **DOM manipulation** (e.g., `document`, `window`)
- **Data fetching / network requests** (e.g., `fetch`, `XMLHttpRequest`)
- **Timers**:
  - `setTimeout()`
  - `setInterval()`
- And more...

### Key idea:
JavaScript uses Web APIs to handle **asynchronous operations** (via the event loop).
